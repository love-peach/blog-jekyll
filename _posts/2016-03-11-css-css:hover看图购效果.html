看图购是一种精美的淘宝网购物导航站点,一个全新的图片网购模式。看图购以图片文集的形式展现淘宝网(目前好像停了)众多丰富的商品!让你快乐购物,享受购物! 看图购目前针对女鞋、女装、内衣、包包、流行配饰、美容美发等类目的宝贝，未来会支持更多的类目。 用户可以通过图片文集,点击购买链接按钮，即可链接到宝贝页面，选购到自己挑选满意的产品，增加购物体验,享受购物乐趣。 其实，这样的需求，纯css就可以做，难怪原作者说要反省，有时候自己确实忽略了太多。

下面是实现的 html 和 css 代码。

{% highlight html linenos%}
{% endhighlight %} {% highlight css linenos %} .ktg { position: relative; width: 960px; } .ktg img { border:4px solid #fff; } .ktg ul { display: none; list-style-type: none; } .ktg:hover ul { display: block; } .ktg ul li { height: 84px; position: absolute; width: 84px; } .ktg ul li div { background: url(http://static.jiawin.com/demos/200/images/borders.png) repeat scroll 0 0 transparent; height: 84px; width: 84px; } .ktg ul li a:hover div { background: url(http://static.jiawin.com/demos/200/images/twoborders.png) repeat scroll 0 0 transparent; height: 84px; width: 84px; } {% endhighlight %}
